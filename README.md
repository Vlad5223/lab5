# Проект: Система управления паттернами проектирования

Проект для изучения и практического применения паттернов проектирования.
Включает реализацию порождающих, структурных и поведенческих паттернов
на языке Python с примерами использования и демонстрацией.

Цель: создать библиотеку паттернов с понятными примерами,
которую можно использовать как учебное пособие.
